# Data Processing Algorithms

Collection of Python tools for dataset validation, metadata extraction, and data quality analysis.

This repository contains reusable scripts designed for data engineering workflows.

## Modules

### Metadata

- metadata_generator → generate metadata for datasets
- metadata_extractor → extract metadata from structured datasets

### Validation

- schema_validator → validate dataset schema
- sensor_validator → validate sensor datasets

### Data Quality

- quality_analyzer → perform data quality analysis
- frequency_analysis → analyze sampling frequency

### Pipelines

- validation_pipeline → end-to-end data processing pipeline

## Technologies

- Python
- Pandas
- Data Engineering workflows
